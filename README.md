# 投掷硬币

这是一个简单的静态网站，用于模拟投掷硬币。用户可以指定要投掷的硬币数量，并在页面上查看每枚硬币翻转后的结果。

## 使用

1. 在页面上输入要投掷的硬币数量。
2. 点击“投掷硬币”按钮。
3. 页面将显示每枚硬币的正面或反面结果，以及正反面的数量。

请注意，此静态网站仅用于演示目的。
